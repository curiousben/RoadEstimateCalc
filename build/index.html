Bristol District L&D Scoping Estimaing Tool

This is a tool dependent on quality engineering estimates of quantities. This tool should NOT be used as a substitute for field oberservation estimate. The overall cost of a project can be modified by adjusting the itemized contingency percentages in the Contingency Tab. Also, the Earthwork Quantity formulas are conservative estimates. If you have any questions regarding the estimating tool contact Chase Buchanan at 276-645-4878

Instructions for this Web Application
  • Contingency % can be adjusted based on your comfort level with item quanity in the Contingnency tab.
  • Unit prices can be adjusted on the Unit prices tab near the receipt tab. Use current unit prices as a guide.
  • CEI % can be adjusted in the Unit prices tabs.
  • Add any additional items that are important in the Misc Information tab.
  • Data input fields with this button shows comments for clarifications.
  • PE or RW cost is not included in this estimate tool.
  • The "Proposed Roadway" header is hyperlinked to the Road Designed Manual.
  • The "Proposed Bridge" header is hyperlinked to Structure and Bridge Vol. V - Part 2 Geometrics.
  • The "Concrete Box Culvert" in the "Drainage" box is hyperlinked to "Box Culvert Computation V4.1".
  • Earthwork inputs will require field data.
